Ibaraki Discovery - Tourisme au Japon

Description

  • David D'AMORE

  • github.com/davistres

  • 2024

Ibaraki Discovery est une plateforme web innovante de promotion du tourisme durable dans la préfecture d'Ibaraki, Japon. Ce projet ambitieux vise à rediriger une partie du flux touristique des destinations surpeuplées (Tokyo, Kyoto) vers cette région authentique située à seulement 1h-1h30 de Tokyo.

Le site propose un système de filtrage avancé (par activités, régions, villes, saisons, accessibilité), une base de données de lieux touristiques et de festivals, et surtout un constructeur d'itinéraire personnalisé intelligent qui génère des circuits sur mesure en fonction des centres d'intérêt, du budget, de la durée et de la saison choisie. L'algorithme prend en compte les floraisons saisonnières, les festivals et événements spécifiques pour proposer l'expérience optimale.

L'application utilise React Router pour une navigation fluide, Framer Motion pour des animations élégantes, et Tailwind CSS pour un design moderne et responsive. Le projet référence une vaste gamme d'activités : nature & aventure (Mont Tsukuba, Hitachi Seaside Park), culture & histoire (Kashima Jingu, Grand Bouddha d'Ushiku), expériences authentiques (ateliers de poterie, dégustation de saké), et détente (onsen, gastronomie locale).

Note technique : Cette application React nécessite un build de production (npm run build) ou un serveur de développement (npm run dev) pour fonctionner. Les captures d'écran ci-dessus présentent les principales fonctionnalités de la plateforme.

Ce projet est toujours en cours de développement actif.
