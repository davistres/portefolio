Ibaraki Discovery - Tourisme au Japon

Description

  • David D'AMORE

  • github.ibaraki

  • 2024

Ibaraki Discovery est une application Web développée en React pour promouvoir le tourisme durable dans la préfecture d'Ibaraki (Japon) et désengorger les destinations surpeuplées.

Cette plateforme se distingue par l'intégration d'un algorithme intelligent générant des circuits sur-mesure (selon le budget, la durée, ou les floraisons saisonnières). Le projet démontre une stack front-end moderne :

  • Architecture React : Développement de composants réutilisables, gestion d'état complexe et routage fluide via React Router.
  • UI/UX Dynamique : Interface moderne et responsive conçue avec Tailwind CSS et animée grâce à Framer Motion.
  • Logique de filtrage : Système de recherche multicritères avancé sur une vaste base de données d'activités touristiques locales.

Technologie / Compétences

  • React 18
  • Vite.js
  • React Router
  • Tailwind CSS
  • Framer Motion
  • Lucide Icons
  • Algorithmique
  • Responsive Design
