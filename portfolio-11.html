EcoRide - Plateforme de Covoiturage Écologique

Description

  • David D'AMORE

  • github.com/davistres

  • 2024

EcoRide est mon projet le plus ambitieux : une plateforme complète de covoiturage écologique inspirée de BlaBlaCar. Développée avec Laravel et Vue.js, cette application full-stack propose un système complet de recherche de trajets, réservations, messagerie en temps réel, gestion de profils utilisateurs, système de notation et d'avis. Le projet inclut une architecture Docker, une API RESTful, un système d'authentification sécurisé, et une interface utilisateur moderne avec Tailwind CSS. L'application a été déployée sur Heroku avec une base de données JawsDB MySQL. C'est une démonstration complète de mes compétences en développement web full-stack.
