EcoRide - Plateforme de Covoiturage Écologique

Description

  • David D'AMORE

  • github.com/davistres

  • 2024

EcoRide est mon projet le plus ambitieux : une plateforme complète de covoiturage écologique inspirée de BlaBlaCar. Développée avec Laravel et Vue.js, cette application full-stack propose un système complet de recherche de trajets, réservations, messagerie en temps réel, gestion de profils utilisateurs, système de notation et d'avis. Le projet inclut une architecture Docker, une API RESTful, un système d'authentification sécurisé, et une interface utilisateur moderne avec Tailwind CSS. C'est une démonstration complète de mes compétences en développement web full-stack.

Note : Ce projet utilise Docker pour son environnement de développement. L'application complète nécessite un backend Laravel et une base de données, elle ne peut donc pas être affichée directement dans un iframe. Les captures d'écran ci-dessus présentent les principales fonctionnalités de la plateforme.
