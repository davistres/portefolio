EcoRide - Plateforme de Covoiturage Écologique

Description

  • David D'AMORE

  • heroku.ecoride

  • 2024

Développement from scratch d'EcoRide, une plateforme complète de covoiturage écologique (concept similaire à BlaBlaCar). Ce projet est ma réalisation la plus ambitieuse à ce jour.

Cette application Full-Stack démontre ma capacité à gérer un projet complexe dans son intégralité, de la conception de la base de données jusqu'au déploiement en production :

  • Stack Technique : Architecture robuste propulsée par Laravel (PHP) en back-end et Vue.js / Tailwind CSS pour une interface utilisateur réactive.
  • Fonctionnalités avancées : Recherche de trajets avec filtres multi-critères, module de réservation, messagerie en temps réel, et système d'avis/notations.
  • Sécurité & API : Développement d'une API RESTful sécurisée et mise en place d'un système d'authentification complet.
  • Déploiement (DevOps) : Environnement développé sous Docker et déployé en ligne via Heroku avec une base de données MySQL (JawsDB).
