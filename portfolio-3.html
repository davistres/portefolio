Série de 5 micro-projets front-end qui illustre ma capacité à concevoir des composants interactifs variés. L'objectif de ce laboratoire technique était d'approfondir la manipulation du DOM, la gestion des états et l'intégration de bibliothèques tierces.

Ces réalisations démontrent une maîtrise croissante du développement web (HTML5, CSS3, JavaScript) à travers différents cas d'usage :

  1. Tableau de données : Intégration avancée et structuration sémantique (HTML/CSS),
  2. Visualisation de données : Intégration graphique interactive avec Chart.js et Plotly.js,
  3. Formulaires : Validation dynamique des saisies utilisateurs côté client (JavaScript),
  4. Sécurité : Conception logique d'un système de captcha interactif,
  5. Gestion d'état : Développement d'une To-Do List complète exploitant le LocalStorage pour la persistance des données.

Chaque exercice illustre des compétences spécifiques en HTML, CSS et/ou en JavaScript.

Technologies / Compétences

  • HTML5
  • CSS3
  • JavaScript ES6
  • Chart.js
  • Plotly.js
  • Manipulation du DOM
  • LocalStorage API
  • Validation de formulaires
