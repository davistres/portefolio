Arcadia - Site Web du Zoo

Description

  • David D'AMORE

  • github.arcadia

  • 2024

Création d'une application Web "Single Page Application" (SPA) développée from scratch pour gérer le site et l'administration du Zoo Arcadia.

Cette application démontre la mise en place d'une architecture front-end avancée et l'intégration de bonnes pratiques de déploiement (DevOps) :

  • Routing JS personnalisé : Système de navigation dynamique côté client pour des transitions instantanées sans rechargement de page.
  • Gestion des accès : Simulation d'un flux d'authentification avec le LocalStorage offrant des interfaces dédiées (Admin, Employé, Vétérinaire... ID et mot de passe respectif: admin/0001, employe/0002 et vet/0003).
  • Conteneurisation (DevOps) : Projet entièrement packagé via Docker et servi de manière optimale par un serveur Nginx.
  • Conception UI/UX : Approche Mobile-First 100% responsive, réalisée sans framework (Vanilla CSS/JS).
