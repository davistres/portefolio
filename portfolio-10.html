Arcadia - Site Web du Zoo

Description

  • David D'AMORE

  • github.com/davistres

  • 2024

Site web complet pour le Zoo Arcadia. Ce projet présente une application web moderne avec un système de routing côté client, permettant une navigation fluide entre les différentes sections : accueil, habitats, services, contact. Le site inclut également des interfaces d'administration pour les employés et vétérinaires, avec un système d'authentification sécurisé. L'interface est entièrement responsive et offre une expérience utilisateur immersive pour découvrir les animaux et habitats du zoo.

Technologie

  • HTML5
  • CSS3
  • JavaScript
  • SPA Routing
  • Responsive Design
