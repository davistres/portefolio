Robbie Lens Photographie

Description

  • David D'AMORE

  • github.robbieLens

  • 2024

Création d'un site vitrine statique (HTML5/CSS3) pour présenter le travail de la photographe fictive "Robbie Lens".

L'objectif de ce projet était de concevoir une interface moderne et professionnelle en mettant l'accent sur les bonnes pratiques du web, notamment :

  • Mise en page responsive : Utilisation avancée de Flexbox et CSS Grid.
  • Qualité du code : Structure HTML sémantique et tableaux de données accessibles.
  • SEO & Performances : Optimisation pour le référencement naturel et traitement des images pour un chargement rapide.
  • Accessibilité (a11y) renforcée : Prise en compte des standards pour rendre le site utilisable par tous.

Technologies / Compétences

  • HTML5
  • CSS3
  • Flexbox & Grid
  • Responsive Design
  • SEO Base
  • Accessibilité (a11y)
  • Optimisation web
