# Mon Portfolio de Projets Web

Bienvenue sur mon portfolio ! Ce dépôt rassemble une collection de projets, d'exercices et de démonstrations que j'ai réalisés pour mettre en pratique et présenter mes compétences en développement web.

## Structure du Projet

-   `index.html`: La page d'accueil principale du portfolio.
-   `portfolio-*.html`: Des pages individuelles détaillant chaque projet.
-   `0/`: Un dossier contenant le code source de tous les mini-projets et exercices.
-   `css/`, `js/`, `images/`: Dossiers contenant les assets globaux (styles, scripts, images) pour le portfolio principal.
-   `docs/`: Contient des documents additionnels, comme mon CV.

## Lancer le Portfolio

La plupart des projets sont statiques et peuvent être visualisés en ouvrant directement les fichiers `.html` dans un navigateur.

Pour une expérience optimale, notamment pour les projets qui pourraient nécessiter des requêtes (API, etc.), il est recommandé de lancer un serveur local. Des scripts sont fournis pour faciliter cela :
-   `START_SERVER.bat`
-   `START_SERVER_NODE.bat`
