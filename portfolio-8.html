Babasaki - Artisans Japonais

Description

  • David D'AMORE

  • github.babasaki

  • 2024

Projet de refonte complète du site web pour "Babasaki", un collectif d'artisans japonais traditionnels.

L'objectif principal était de moderniser leur présence en ligne tout en respectant scrupuleusement leur charte graphique et l'esthétique minimaliste propre à la culture japonaise. Ce projet m'a permis de travailler sur toute la stack (du design au développement) :

  • Conception UI/UX : Création d'une interface élégante et épurée, mettant en valeur leur savoir-faire artisanal à travers des visuels forts (comme le Hero Slider interactif).
  • Intégration moderne : Développement front-end 100% responsive et performant en utilisant Tailwind CSS pour un design sur-mesure.
  • Dynamisme & Interactivité : Utilisation de JavaScript pour des transitions douces et de PHP pour la gestion de certaines logiques côté serveur.

Technologie / Compétences

  • HTML5
  • Tailwind CSS
  • JavaScript ES6
  • PHP
  • UI/UX Design
  • Responsive Design
  • Animations fluides
